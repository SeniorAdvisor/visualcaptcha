# Visualcaptcha

This Gem is ruby version of emotionLoop's PHP Visual Captcha

## Installation

Add this line to your application's Gemfile:

    gem 'visualcaptcha'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install visualcaptcha
